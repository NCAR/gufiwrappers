Welcome to gufiwrappers’s documentation!¶

Currently this tool is restricted to be used on Casper only. In order to use this tool do the following:

  1. ssh to casper.ucar.edu (or execdav from cheyenne login nodes)

  2. load gufiwrappers module module load gufiwrappers

  3. run gcache (with appropriate options)

  4. optionally run grprt (to generate brief reports)

  5. optionally run gplot (work in progress) to produce graphical reports

The gcache spawns GUFI query to squall1 and stores the output into <gufi_tmp> tree. The grprt reads the raw report generated by gcache and generates more useful statitics. The gplot will generate graphical representation of some of these statistics. So, the tool set is designed to be run in sequence:

  1. gcache

  2. grprt

  3. gplot

The gufiwrappers github page is here. Please post any bugs, feedbacks, comments or feature requests here or just send an email to me.
